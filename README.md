# python-console-games

[![Maintainability](https://api.codeclimate.com/v1/badges/ae17bf4c1c0bfe2bd2b3/maintainability)](https://codeclimate.com/github/vevdokimovm/Console_Games/maintainability)

A collection of interactive math-based console games written in Python.

## Games

| Game | Description |
|------|-------------|
| **LCM** (`nok`) | Guess the Least Common Multiple of two random numbers |
| **Geometric Progression** (`progression`) | Find the missing element in a geometric sequence |

## Getting Started

```bash
git clone https://github.com/vevdokimovm/python-console-games.git
cd python-console-games
python main.py
```

Or launch a specific game directly:

```bash
python main.py nok
python main.py progression
```
